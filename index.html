Lista de compras

Esta pequeña aplicacion le permitirá crear una lista de productos por id, nombre y precio.

Podrá realizar acciones como:

  • Agrear un producto a la lista (crearlo y agregarlo)
  • Pedir el listado actual
  • Borrar un producto de la lista
  • Modificar los datos de un producto existente
